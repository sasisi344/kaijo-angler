# 魚種別攻略記事の格納

## 魚の例

### メイン
- マダイ（真鯛）
- ブリ系（青物）
    - ワラサ（メジロ）、ブリ、カンパチ、ヒラマサ。

### 高級魚
- シマアジ
- イシダイ（石鯛）
- スズキ（シーバス）

### 季節・特定の施設で人気
- トラフグ（冬季）
- クエ
- コブダイ
- サーモン（冬季）
    - 銀鮭、トラウトサーモンなど

## タグ付けルール
Astroの仕様で魚種はカテゴリ制御よりもタグで拾う感じが妥当。上から人気順になるので優先度は上から。施設紹介に入っている魚種名はすべてフォローして、検索や記事チョイスで選択しやすいようインデックス化することを目指す。
