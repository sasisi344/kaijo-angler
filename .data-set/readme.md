# 📂 海上アングラー データセット一覧

このフォルダには、「海上釣り堀」ジャンルに特化したサイト構築・運営のための市場データ、記事作成計画、アフィリエイト戦略等の資料が格納されています。

## 📊 市場調査・分析レポート

市場の現状、成長予測、利用者層の分析など、サイトの方向性を決定づける根拠データです。

- **[[【DS】海上釣り堀旅行市場レポート【2025年版】]]**
  - 全国79施設の調査に基づく包括的なレポート。地域別分布、旅行業界との連携可能性、SWOT分析などが含まれます。
- **[[海上釣り堀市場成長シナリオ（2025-2030）]]**
  - 2025年から2030年までの市場成長予測（保守・標準・積極の3シナリオ）と、成長のための重点施策。
- **[[海上釣り堀利用者層・ニーズ分析]]**
  - 4つの主要ターゲット（本格派、ファミリー、観光層、団体）の属性、ニーズ、消費単価の分析。
- **[[海上釣り堀 地域別料金分析]]**
  - 高級・中級・格安の価格帯別施設リストと、料金格差の要因分析。
- **[[【DS】海上釣り堀_地域別釣法・ブランド魚データベース]]**
  - 瀬戸内（オリーブハマチ）、三重、関東など地域ごとの特徴やブランド魚、ローカルルールを網羅。
- **[[【DS】海上釣り堀タックル・ギア需給ギャップ分析]]**
  - ユーザーの不満（竿の破損、リールトラブル等）と市場供給のギャップを分析し、最適なギアと記事テーマを提案。

## 📝 記事作成計画・ガイドライン

具体的な記事のネタ帳、構成案、進捗管理用のリストです。

- **[[海上釣り堀で釣れる魚のリストと記事作成計画]]**
  - 魚種別（高級魚、中型魚、イカ・タコ等）の記事作成リスト。SEOタグやターゲット層の設定も記載。
- **[[【DS】海上釣り堀_科学的エビデンス・理論データ]]**
  - 水温、代謝、水中光学、アミノ酸刺激、アクション理論などの科学的根拠を体系化。
- **[[【DS】海上釣り堀_魚の持ち帰り・熟成・調理データ]]**
  - 究極の血抜き（津本式）の効果、熟成（エイジング）の至適条件、家庭での温度管理法。
- **[[【DS】海上釣り堀_必勝釣法・テクニック体系化データ]]**
  - ウキ釣り・ミャク釣り・ズボ釣りの比較と状況別使い分け、食い渋り時の具体的戦略を体系化。
- **[[【DS】海上釣り堀ユーザー悩み・解決策データベース]]**
  - ユーザーが抱える具体的悩み（技術、環境、人間関係）と、それに対する解決策・記事アイデアのデータベース。
