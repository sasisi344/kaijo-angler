# 香川・小豆島エリア 交通調査報告（海上釣り堀・海釣り施設利用者向け）

## A. エリア全体の交通マップ

小豆島へのアクセスはフェリーが中心となり、出発地（高松・岡山・姫路・神戸）によって到着港が異なります。

| 項目 | 自動車（フェリー航送） | 旅客のみ（フェリー・高速船） |
| :--- | :--- | :--- |
| **最短ルート** | 高松港〜土生港（60分）/ 岡山港〜土庄港（70分） | 高松港〜土庄港（高速船35分） |
| **コスト優先** | 岡山・高松航路（距離が短いため） | 旅客運賃のみ（約700円〜） |
| **運転負荷** | 低（船内で休息可能） | 極低（公共交通機関のみ） |
| **車なし可否** | 推奨（島内の移動に必要） | 可能（港からバス・タクシー・送迎） |
| **読者像** | **家族・グループ**（荷物多め） | **単身・ビギナー**（手ぶら派） |

## B. フェリー・高速船（主要航路）

### 車両航送料金（5m未満・片道概算）
*   **高松港 〜 土庄港**（四国フェリー）：約6,000円〜 [1]
*   **新岡山港 〜 土庄港**（四国フェリー・小豆島フェリー）：約6,000円〜 [1]
*   **姫路港 〜 福田港**（小豆島フェリー）：約10,000円〜 [1]
*   **神戸港 〜 坂手港**（ジャンボフェリー）：約10,000円〜 [2]
*   **注意点**：往復割引（復路1割引）や、ジャンボフェリーの平日割引・深夜料金設定に留意。

### 釣り具持ち込み規定
*   **フェリー**：旅客のみの場合、手回り品として持ち込み可能。大型クーラーボックスや竿ケースは、混雑状況によりデッキ等への置き場所を指定される場合がある。
*   **高速船**：スペースが限られるため、長尺物（ロッドケース）は事前に持ち込み可否を確認するか、コンパクトなパックロッドを推奨。

## C. 島内の二次交通（ラストマイル）

1.  **路線バス（小豆島オリーブバス）**：主要港（土庄、池田、草壁）から島内各所を結ぶ。
    *   **荷物制限**：他のお客様の迷惑にならない範囲。大型クーラーボックスの持ち込みは混雑時は避けるのがマナー。
2.  **レンタカー**：土庄港、池田港周辺に営業所あり。釣り具の汚れ対策（ブルーシート等）は必須。
